# -Meu-Primeiro-Copiloto-com-Microsoft-Copilot-Studio

O objetivo principal foi explorar a ferramenta e construir um agente personalizado, entendendo como configurar fluxos de conversa, definir respostas e criar um agente útil para resolver um problema real.

##  O que foi feito

Durante a prática:

- Acompanhei a **demonstração oficial** do Microsoft Copilot Studio.
- Aprendi a **criar um copiloto** com funcionalidades básicas, incluindo:
  - Definição de propósito do agente.
  - Criação de fluxos de conversa.
  - Configuração de respostas personalizadas.
  - Ajuste do tom de voz e regras de interação.
- Desenvolvi o **Agente de Viagens Seguras I.F**, voltado para auxiliar **brasileiros** na verificação de **exigências de visto internacional**.

## Sobre o Agente Criado

**Nome:** Agente de Viagens Seguras I.F  
**Função:** Consultar em tempo real as exigências de visto para brasileiros no exterior.  
**Principais recursos:**
- Verificação de exigência ou isenção de visto por país.
- Identificação do tipo de visto conforme o motivo da viagem (turismo, negócios, estudo etc.).
- Lista de documentos exigidos, prazos e validade.
- Indicação de fontes oficiais (sites governamentais, embaixadas e consulados).
